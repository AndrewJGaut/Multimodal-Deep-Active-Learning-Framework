# CS229-Project

## Directory structure

```
├── README.md          <- The top-level README for developers using this project.
├── data
│   └── kaggle_satellite_image_classification   <- Toy dataset, downloaded from
│                                                  https://www.kaggle.com/mahmoudreda55/satellite-image-classification
│
├── environment.yml    <- The requirements file for reproducing the analysis environment,
│                         generated with `conda env export > environment.yml`,
│                         loaded with `conda env create -f environment.yml`, and
│                         updated with `conda env update --name cs229proj --file environment.yml --prune`
│
```
